Summary

The global layout provides vertical arrangement of page content to provide support for three navigation location options.

  1. Left Navigation (recommended)
  2. Content Only (no navigation)
  3. Top Navigation (discouraged)

Left Navigation

Content-Only

For cases where your application may be embedded without the need for a navigation, use the "Content-Only" layout.

Top Navigation

Warning HelixUI does not provide styling for Top Navigation layouts.

You will be responsible for maintaining the appearance of a Top Navigation bar.

Please review the design specifications for more information.

Eyebrow

The eyebrow will be where the Pilot application lives for navigation between Helix applications.
