Columns

  • Columns are containers that allow you to divide their height amongst their children.
  • Columns are the basis for all sectioned content within a Row.
  • Columns share width equally among other Columns in the same Row.

Column Width

Column width modifiers will set the width of a Column container to a predefined percentage of the Row width.

Use the .hxCol-{N} modifier (1 ≤ N ≤ 12) to create columns that are a fraction of the 12-column row width.

  • .hxCol-{N}-{breakpoint} responsive classes are available
  • .hxCol-{N} is shorthand for .hxCol-{N}-xs

Column Ordering

Use the .hxOrder-{N} classes (1 ≤ N ≤ 12) to reorder your columns.

  • .hxOrder-{N}-{breakpoint} responsive classes are available
  • .hxOrder-{N} is shorthand for .hxOrder-{N}-xs

Responsive Options

  • All column modifiers support adding an optional breakpoint.
  • Omitting the breakpoint will default xs (applies to all breakpoints).
  • Each responsive class corresponds to the minimum breakpoint required to achieve the desired style.
  • Larger breakpoints override smaller ones.
CSS Class Minimum Breakpoint Overrides Overridden By
.hxClass-{N}[-xs] 0em (0px) N/A all below
.hxClass-{N}-sm 40em (~640px) all above all below
.hxClass-{N}-md 64em (~1024px) all above all below
.hxClass-{N}-lg 75em (~1200px) all above all below
.hxClass-{N}-xl 90em (~1440px) all above N/A

Styling Containers

Warning Do not attempt to style Rows or Columns in a Grid.
  • Grids have margins set to align the outer Columns with surrounding content.
  • Columns in a Grid have padding applied to them to simulate gutters.
  • Wrap your content in a .hxContainer to apply styles using custom classes.
