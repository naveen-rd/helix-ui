Vertical Layout

  • Use the .hxApp class on <body>.
  • Use this layout if your application content needs to grow vertically (pushing the footer down).
  • .hxApp__content will arrange its contents in a column (normal document flow) with padding applied.
  • *Vertical Layout Template

Horizontal Layout

  • Use the .hxApp--horizontal class on <body>.
  • Use this layout if your application content needs to grow horizontally (keeping the footer visible at all times).
  • .hxApp__content will arrange its contents in a row without padding.
  • *Horizontal Layout Template

CSS Class Reference

.hxApp
Use on the <body> element to achieve a vertical application layout.
This is used when you need content to grow vertically.
.hxApp--horizontal
Use on the <body> element to achieve a horizontal application layout.
This is used when you need content to grow horizontally (e.g. for a two column layout).
.hxApp__header
Use on a <header> element that is a child of <body> to apply styles for the Eyebrow.
Pilot lives in this element.
.hxApp__footer
Use on a <footer> element that is a child of <body> to apply styles for the global footer.
.hxApp__stage
Use on a <div> element between .hxApp__header and .hxApp__footer.
.hxApp__content
Use on a <main> element within .hxApp__stage to house your application content.
